# 1. Core Gameplay Loop

## 1.1 Loop Steps

The basic cycle the player repeats for the entire playtime:

- Enter an area - everything dark, threatening, drained of color
- Explore, leave color behind through movement
- Fight enemies - dynamic melee/ranged decision (both available from the start)
- Find a Secret Room, pay with Shield/Prisma to open it - permanent character buff
- Master a level's gimmick/special mechanic
- Defeat a Miniboss / Templateboss / Chapterboss
- Color explosion in the level - the area comes back to life in color, a Token is unlocked (Templateboss/Chapterboss additionally: a Merchant appears)
- Optional: trade Tokens with the Merchant for a new ability
- Reach the exit portal/level end - back to the Worldmap
- Worldmap: the node and surrounding path turn colorful
- A new area opens up

## 1.2 Context

This loop applies unchanged to the Prologue and all Chapters - only scale and enemy strength change. Details on the Template structure in [02_game-structure.md](02_game-structure.md), on the Token/Buff economy also there (2.5).
