# 8. UI & HUD

| Element | Position | Details |
|---|---|---|
| Health Bar | Top left | Numeric health bar, see [04_health-save-system.md](04_health-save-system.md) 5.1 |
| Prisma Bar | Top left, next to Health Bar | Shield display, see [03_mechanics.md](03_mechanics.md) 4.5 |
| Chapter/Level | Top right | Progress display |
| Boss HP Bar | Top center | Large, dramatic - only during boss fights |
| Ability Popup | Center of screen | On unlock + short tutorial |
| Left/Right/Duck Button | Bottom left | Mobile only |
| Jump/Attack Button | Bottom right | Mobile only, attack with auto-targeting of the nearest enemy |

No score. Progress is measured through colored areas, defeated bosses, and chapter status.
